En mi proyecto final del simulador de autos, me permite gestionar una lista de modelos de autos a través del almacenamiento por localStorage. El usuario puede agregar, visualizar y eliminar el modelo del auto por una lista en la cual interactúa. También integro una API externa para obtener informaciones sobre otras marcas de autos.

La organización del proyecto se encuentra por carpetas, una donde tiene el nombre llamado "Classes" ahí están las clases de los modelos de los vehículos y facturas de compra (car.js y facturacompra.js).
La carpeta llamada "Scripts" donde se encuentran las constantes (constantes.js) y el mas importante "Main" (main.js)
También cree una carpeta de estilos (Styles) "style.css" donde le doy un poco de estilos a mi proyecto.

El proyecto cuenta con variables, arrays, objetos, manejo de DOM, eventos, almacenamiento en el localStorage, recuperación de datos desde el storage.
También cuenta con una API externa donde uso fetch() para obtener los datos de la misma.
